# Numis Nepal API

This api connects with mongodb atlas to store data.

To connect with mongodb atlas assign mongodb connecting string to  `MONGO_URI` in `.env` file.

Mongo uri structure 
`mongodb+srv://<username>:<password>@<cluster>.czh3y.mongodb.net/<databasename>?retryWrites=true&w=majority`


To run, enter command
### `npm i`
### `npm run server`
